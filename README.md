# 🚀 ToolHubAI

### AI-Powered Productivity Toolkit

ToolHubAI is a modern AI-powered web application that combines multiple useful tools into one platform, including image editing, document conversion, resume analysis, and AI enhancement utilities.

---

## ✨ Features

✅ **Background Remover**
Remove image backgrounds instantly using AI.

✅ **PDF → DOCX Converter**
Convert PDF files into editable Word documents.

✅ **DOCX → PDF Converter**
Convert Word documents into PDF format.

✅ **AI Resume Analyzer**
Get resume insights, ATS suggestions, skill detection, and improvement recommendations.

✅ **AI Image Enhancer**
Enhance image quality for sharper visuals.

✅ **AI Photo Upscaler**
Increase image resolution while preserving quality.

✅ **Object Remover** *(New)*
Remove unwanted objects from images.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Tarique2002/toolhubai.git
cd toolhubai
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

### Backend Setup

```bash
cd backend
python -m venv venv
venv\Scripts\activate

pip install -r requirements.txt
python -m uvicorn main:app --reload
```
