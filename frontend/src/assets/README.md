# 🚀 ToolHubAI

ToolHubAI is an all-in-one AI-powered productivity platform built to simplify common digital tasks such as image editing, file conversion, and resume analysis — all in one place.

## ✨ Features

### 🖼 Background Remover

Remove image backgrounds instantly using AI.

### 📄 PDF → DOCX Converter

Convert PDF files into editable Word documents.

### 📝 DOCX → PDF Converter

Convert Word documents into PDF format.

### ✨ AI Image Enhancer

Enhance image quality and sharpness.

### 🔥 AI Photo Upscaler

Upscale low-quality images for better clarity.

### ⭐ AI Resume Analyzer

Analyze resumes and get:

* Resume score
* Skill detection
* ATS suggestions
* Improvement recommendations

### 🧹 Object Remover *(In Progress)*

Remove unwanted objects from images.

## ⚙️ Installation

### 1. Clone Repository

```bash
git clone https://github.com/Tarique2002/toolhubai.git
cd toolhubai
```

### 2. Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

### 3. Backend Setup

```bash
cd backend

python -m venv venv

# Activate venv
venv\Scripts\activate

pip install -r requirements.txt

python -m uvicorn main:app --reload
```
